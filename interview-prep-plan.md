Good task examples:

* Solve the exact coding problem “Two Sum” using a dictionary. Then explain why the brute-force approach is `O(n²)` and the dictionary approach is `O(n)`.
* Design a tiny in-memory cache with `Get`, `Set`, TTL expiry, and max-size eviction.
* Create a RabbitMQ Docker Compose file with one exchange, two queues, and one dead-letter exchange.
* Build an Azure Service Bus sender and receiver console app using queue messages.
