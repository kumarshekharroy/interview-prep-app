## Launch Criteria

Start high-priority applications when all are true:

* [ ] Resume is final enough to send.
* [ ] LinkedIn is aligned with resume.
* [ ] At least 20 target roles are tracked.
* [ ] At least 10 referral targets are identified.
* [ ] Recruiter templates are ready.
* [ ] DSA final retake score is passing.
* [ ] System design final retake score is passing.
* [ ] Project defense is passing.
* [ ] Behavioral mock is passing.
* [ ] Salary scripts are ready.

## Interview-Day Checklist

Use before every scheduled interview.

* [ ] Read job description.
* [ ] Identify likely interview focus.
* [ ] Review company notes.
* [ ] Review target role fit answer.
* [ ] Review two project stories.
* [ ] Review one DSA pattern.
* [ ] Review one system design prompt.
* [ ] Prepare five questions.
* [ ] Test laptop, camera, microphone, internet.
* [ ] Keep resume and notes nearby.
* [ ] Plan water, timing, and quiet environment.

## Post-Interview Checklist

Complete immediately after each interview.

* [ ] Write questions asked.
* [ ] Score your performance.
* [ ] Identify weak answers.
* [ ] Add weak areas to weak-area log.
* [ ] Send thank-you note if appropriate.
* [ ] Update application tracker.
* [ ] Schedule retake before next round.
